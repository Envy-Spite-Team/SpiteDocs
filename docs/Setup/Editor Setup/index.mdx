# Editor Setup

Rude Level Editor is created by Vanity Reprise program and is based on Unity 2022. In this page you will learns how to install Unity, Vanity Reprise and Rude.

# Downloading Unity Hub

Unity Hub is used for launching Unity projects such as Rude and is required in order to work with Unity Engine. Here are the direct download links for Unity Hub:
- <b>[For Windows](https://public-cdn.cloud.unity3d.com/hub/prod/UnityHubSetup.exe)</b>
- <b>[For MacOS](https://public-cdn.cloud.unity3d.com/hub/prod/UnityHubSetup.dmg)</b>
- <b>[For Linux](https://docs.unity3d.com/hub/manual/InstallHub.html#install-hub-linux)</b>

Follow the instructions given in the installer program and it will be ready to use.

:::warning
Unity Hub will ask you to install Unity 6 after its first launch. Decline if you don't plan to do anything with it.
:::

# Installing Unity 2022

In order to install Unity 2022 firstly, press <b>[this direct link](unityhub://2022.3.62f1/4af31df58517)</b> and then permit this website to open Unity Hub. It should automatically start installing it. This process can take an hour so you can proceed to the next step.

# Installing and using Vanity Reprise

:::warning
Vanity Reprise extracts a lot of ULTRAKILL assets and is heavy on low-end PCs.
It is recommended to have between 9-12 GB of storage on your PC.
:::

Head over to the <b>[latest release page for Vanity Reprise](https://github.com/eternalUnion/VanityReprised/releases/latest)</b>, and download VanityReprised-1.x.x.zip file.

<div style={{ textAlign: "center" }}>
    <img
        src={require('./assets/vanity-github.png').default}
        alt="Vanity GitHub Page"
    />
</div>

Extract the zip file into a folder and run Vanity.GUI.Free.exe.

<div style={{ textAlign: "center" }}>
    <img src={require('./assets/vanity-files.png').default} alt="Vanity.exe" width="50%" height="50%" ></img>
</div>

A Command Prompt should open up and you should see a new webpage in your browser. In that webpage you can select specific scenes to export from the game by navigating with arrows. It is advised to export a couple of scenes such as P-2 and 1-3 in order to be able to see how official ULTRAKILL levels have been created. You are free to choose more scenes however, it will take more time and storage space to export.

<div style={{ textAlign: "center" }}>
    <img src={require('./assets/vanity-website.png').default} alt="Vanity Website" width="80%" height="80%" ></img>
</div>

Below, there is a `Open ULTRAKILL folder` button. Press it and select the location of you ULTRAKILL installation. You can find out where it is located by viewing local files in Steam. After that, select where you want to install your Rude Level Editor(it is highly recommended to create a separate folder in your non-system drive(not C:\) and select that). Now, wait until you see a message about finished export.

# Post-Extraction

Once Vanity Reprise is run, open Unity Hub, click the little arrow next to the "Add" Button, click "Add from disk" and select the location of where you chose to install the editor. After that, you can finally open your level Editor!.

:::danger
When doing so you might be prompted with a window asking to update backends and API related stuff, **Always make 
sure to say no to these popups when opening Rude.** Otherwise, it will break your Editor.
:::
